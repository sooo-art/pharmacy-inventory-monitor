# Pharmacy Inventory Health Monitor

## Project Overview

This project simulates a pharmacy inventory monitoring system designed to identify stock-out risks, monitor inventory health, and support data-driven replenishment decisions.

The project combines SQL, Python, and Power BI to transform raw inventory data into actionable business insights.

This project was inspired by real-world inventory management and stock replenishment processes observed in a high-volume pharmacy retail environment.

---

## Business Problem

Retail pharmacies manage hundreds of SKUs every day.

Without effective inventory monitoring:

- High-demand products may go out of stock
- Slow-moving products may become overstocked
- Inventory holding costs can increase
- Customer satisfaction can decline

The objective of this project is to build an inventory health monitoring solution that helps identify:

- Reorder alerts
- Fast-moving products
- Slow-moving products
- Stock-out risks
- Inventory turnover performance
